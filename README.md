## Calculadora em Java

Este projeto foi desenvolvido como trabalho prático da disciplina de Java da faculdade.

A aplicação consiste em uma calculadora com interface gráfica, onde foram implementadas operações matemáticas básicas.

## Tecnologias utilizadas
- Java
- NetBeans

## Objetivo
Praticar os conceitos básicos de programação em Java, incluindo lógica de programação e desenvolvimento de interfaces gráficas.
